DMZ Lua API Reference Guide

dmz.vector

The dmz.vector class provides bindings for the dmz::Vector class.

dmz.vector.new (x, y, z)

dmz.vector.new (value)

Creates a new dmz.vector. The parameter value may either be a dmz.vector or an array with three numbers. The parameters x, y, and z are numbers.

dmz.vector.is_a (value)

Tests if value is a dmz.vector. Returns a dmz.vector if true and nil if false.

dmz.vector:set_x (value)

Sets the x value of the dmz.vector. Returns a reference to self.

dmz.vector:set_y (value)

Sets the y value of the dmz.vector. Returns a reference to self.

dmz.vector:set_z (value)

Sets the z value of the dmz.vector. Returns a reference to self.

dmz.vector:set_xyz (x, y, z)

Sets the x, y, and z components of the dmz.vector. Returns a reference to self.

dmz.vector:get_x ()

Gets the x value of the dmz.vector. Returns a number.

dmz.vector:get_y ()

Gets the y value of the dmz.vector. Returns a number.

dmz.vector:get_z ()

Gets the z value of the dmz.vector. Returns a number.

dmz.vector:get_xyz ()

Gets the x, y, and z components of the dmz.vector. Returns three numbers.

dmz.vector:magnitude ()

Calculates the magnitude of the vector. Returns a number.

dmz.vector:normalize ()

Creates and returns a normalized version of the vector.

dmz.vector:dot (value)

Calculates and returns the dot product of the dmz.vector and value. The parameter value must evaluate to a dmz.vector.

dmz.vector:cross (value)

Calculates and returns the cross product of the dmz.vector and value. The parameter value must evaluate to a dmz.vector. Returns a dmz.vector.

dmz.vector:is_zero ([epsilon])

Tests if the x, y, and z components of the dmz.vector are zero. Returns true if the all components are zero. It will otherwise return false. The optional parameter epsilon specifies the error to allow when calculating the value of zero.

dmz.vector:get_angle (value)

Calculates and returns the angle between the dmz.vector and value. The angle is returned as a number in radians. The parameter value must evaluate to a dmz.vector.

dmz.matrix

The dmz.matrix class provides bindings for the dmz::Matrix class.

dmz.matrix.new (v1, v2, v3, v4, v5, v6, v7, v8, v9)

dmz.matrix.new (value)

Creates a new dmz.matrix. The parameter value may either be a dmz.matrix or an array with nine numbers. The parameters v1 through v9 are numbers.

dmz.matrix.is_a (value)

Tests if value is a dmz.matrix. Returns a dmz.matrix if value is a dmz.matrix and nil if it is not.

dmz.matrix:transpose ()

Creates and returns a dmz.matrix that is a transpose of the current dmz.matrix.

dmz.matrix:invert ()

Creates and returns a dmz.matrix that is an inverse of the current dmz.matrix.

dmz.matrix:transform (value)

Creates and returns a dmz.vector that is the transformed value. The parameter value must evaluate to a dmz.vector.

dmz.matrix:from_table (value)

Updates the dmz.matrix to use the array of numbers stored in value. The parameter value is an array of nine numbers. A reference to self is returned.

dmz.matrix:to_table ()

The nine components of the dmz.matrix are returned in an array.

dmz.matrix:from_two_vectors (fromVec, toVec)

Use the fromVec and toVec to compose new values for the dmz.matrix. The parameters fromVec and toVec must evaluate to dmz.vectors. A reference to self is returned.

dmz.matrix:from_axis_and_angle (axis, angle)

Use the axis and angle to compose new values for the dmz.matrix. The parameter axis must evaluate to a dmz.vector. The parameter angle is a number. A reference to self is returned.

dmz.matrix:is_identity ()

Test if the dmz.matrix is an identity matrix. Returns true if it is an identity matrix.

dmz.matrix:set_identity ()

Coverts the matrix to an identity matrix. A reference to self is returned.

dmz.mask

The dmz.mask class provides bindings for the dmz::Mask class.

dmz.mask.new ([value])

Creates and returns a new dmz.mask. The optional value may be a dmz.mask which is used to initialized the new dmz.mask.

dmz.mask.is_a (value)

Tests if value is a dmz.mask. Returns a dmz.mask if value is a dmz.mask and nil if it is not.

dmz.mask:clear ()

Clears the mask. Returns a reference to self.

dmz.mask:contains (value)

Test if the dmz.mask contains value. The parameter value must evaluate to a dmz.mask. Returns true if the mask contains value.

dmz.mask:unset (value)

The dmz.mask has all bits defined in value unset. The parameter must evaluate to a dmz.mask. Returns true if the mask is successfully unset.

dmz.mask:bit_and (value)

Creates and returns a dmz.mask that is a bitwise and of the dmz.mask and value. The parameter value must evaluate to a dmz.mask.

dmz.mask:bit_or (value)

Creates and returns a dmz.mask that is a bitwise or of the dmz.mask and value. The parameter value must evaluate to a dmz.mask.

dmz.mask:bit_xor (value)

Creates and returns a dmz.mask that is a bitwise exclusive-or (i.e. xor) of the dmz.mask and value. The parameter value must evaluate to a dmz.mask.

dmz.mask:bit_set (value)

Sets the bit specified by value. The parameter value is a number. Returns a reference to self.

dmz.mask:bit_unset (value)

Unsets the bit specified by value. The parameter value is a number. Returns a reference to self.

dmz.mask:bit_not ()

Creates and returns a mask that is the bitwise not of the dmz.mask.

dmz.mask:bit_shift (shift)

Shifts the mask. A positive value shifts up while a negative value shifts down. Returns a reference to self.

dmz.handle

The dmz.handle class provides bindings for the dmz::Handle type. Handles are atomic and maybe be used as key values in tables.

dmz.handle.new (value)

Creates a dmz.handle. The parameter value must either be a string or number. If value is a string, a named handle is created. If value is a number, the dmz.handle is initialized with the number.

dmz.handle.is_a (value)

Tests if value is a dmz.handle. Returns a dmz.handle if value is a dmz.handle and returns nil if it is not.

dmz.uuid

The dmz.uuid class provides bindings for the dmz::UUID class. UUIDs are atomic and maybe be used as key values in tables.

dmz.uuid.new ()

Creates a new dmz.uuid. The newly created dmz.uuid is initialized with a unique UUID.

dmz.uuid.is_a (value)

Tests if value is a dmz.uuid. Returns a dmz.uuid if value is a dmz.uuid and returns nil if it is not.

dmz.config

The dmz.config class provides bindings for the dmz::Config class.

dmz.config.get_global ()

Returns a dmz.config object containing the global config data.

dmz.config.new ([name])

Returns a new dmz.config object. The name is used to initalize the new dmz.config object's tag.

dmz.config.is_a (value)

Returns a dmz.config if value is a dmz.config object. Returns nil if it is not.

dmz.config:get_name ()

Returns the name of the dmz.config object. Returns nil if the dmz.config object is empty.

dmz.config:get_attributes ()

Returns a table containing the attributes stored in the dmz.config object. Returns nil if the dmz.config object is empty.

dmz.config:store_attribute (name, value)

Store an attribute of name with value in the dmz.config object. Returns true if the attribute was successfully stored.

dmz.config:lookup_attribute (name)

Returns the value of the attribute name. Returns nil if the dmz.config object does not contain an attribute with the given name.

dmz.config:lookup_boolean (name [, default])

Looks up the attribute value name and returns it as a boolean. If the attribute name is not found, then default is returned.

dmz.config:lookup_string (name [, default])

Looks up the attribute value name and returns it as a string. If the attribute name is not found, then default is returned.

dmz.config:lookup_number (name [, default])

Looks up the attribute value name and returns it as a number. If the attribute name is not found, then default is returned.

dmz.config:lookup_handle (name [, default])

Looks up the attribute value name and returns it as a dmz.handle. If the attribute name is not found, then default is returned.

dmz.config:lookup_matrix (name [, default])

Looks up the attribute value name and returns it as a dmz.matrix. If the attribute name is not found, then default is returned.

dmz.config:lookup_event_type (name [, default])

Looks up the attribute value name and returns it as a dmz.event_type. If the attribute name is not found, then default is returned. Returns nil if a dmz.event_type can not be found.

dmz.config:lookup_message (name [, default])

Looks up the attribute value name and returns it as a dmz.message. If the attribute name is not found, then default is returned. Returns nil if a dmz.message can not be found.

dmz.config:lookup_object_type (name [, default])

Looks up the attribute value name and returns it as a dmz.object_type. If the attribute name is not found, then default is returned. Returns nil if a dmz.object_type can not be found.

dmz.config:lookup_uuid (name [, default])

Looks up the attribute value name and returns it as a dmz.uuid. If the attribute name is not found, then default is returned.

dmz.config:lookup_vector (name [, default])

Looks up the attribute value name and returns it as a dmz.vector. If the attribute name is not found, then default is returned.

dmz.input_observer

An event is stored as a table and has the following structure: 
{
   axis = {
      source, -- dmz.handle
      which, -- number
      value, -- number
      previous_value, -- number
      delta, -- number
   },
   button = {
      source, -- dmz.handle
      which, -- number
      value, -- boolean
      previous_value, -- boolean
      changed, -- boolean
   },
   switch = {
      source, -- dmz.handle
      which, -- number
      value, -- number
      previous_value, -- number
      changed, -- boolean
   },
   key = {
      source, -- dmz.handle
      value, -- number
      state, -- boolean
   },
   mouse = {
      source, -- dmz.handle
      window_size_x, -- number
      window_size_y, -- number
      x, -- number
      y, -- number
      previous_x, -- number
      previous_y, -- number
      delta_x, -- number
      delta_y, -- number
      scroll_delta_x, -- number
      scroll_delta_y, -- number
      buttons, -- dmz.mask
      previous_buttons, -- dmz.mask
   },
}
The input callback should accept two parameters: self and an event. 
   function input_callback (self, event)
      if event.button then -- process mouse event
      elseif event.key then -- process key event
      elseif event.mouse then -- process mouse event
      end
   end
